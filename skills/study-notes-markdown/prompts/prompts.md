# study-notes-markdown prompts

這份檔案提供常用提示詞範例，讓使用者更容易明確喚起 `study-notes-markdown`。

重點不是要求模型重寫內容，而是要求它保留教材型正文，補上結構、導航、查找入口與複習支架。

---

## 使用方式

- 直接把下面的句子改掉檔名、資料夾名或任務範圍即可
- 若你知道自己更重視什麼，可明講：例如保留正文、先拆分、補查找入口、補複習支架、同步 README
- 若你不確定要不要拆分，可直接要求它依 skill 規則判斷，並在收尾時說明理由
